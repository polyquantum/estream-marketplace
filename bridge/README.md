# polyquantum.bridge

Cross-chain and cross-lex bridge circuits for eStream interoperability.

## Modules

- **cross_chain** — Bridge circuits to external blockchains (Ethereum, Solana, etc.)
- **cross_lex** — Inter-lex bridges for sovereign lex domain interoperability
- **relay** — Light-client relay proofs for trustless cross-chain verification

## Platform Requirements

- eStream platform >= 0.12.0
- ESCIR API 1.0.0
