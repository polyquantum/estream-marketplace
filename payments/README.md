# polyquantum.payments

Payment protocol circuits for the eStream marketplace.

## Modules

- **iso20022** — ISO 20022 (SWIFT MX) message parsing, validation, and transformation
- **htlc** — Hash Time-Locked Contract circuits for atomic cross-chain swaps
- **offline** — Offline payment verification with delayed settlement sync
- **settlement** — Settlement DAG for multi-party clearing and netting

## Platform Requirements

- eStream platform >= 0.12.0
- FLIR API 1.0.0
