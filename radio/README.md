# polyquantum.radio

Radio spectrum management and tactical mesh networking circuits for eStream.

## Modules

- **spectrum** — 5G/6G dynamic spectrum allocation and sharing with ZK-verified entitlements
- **mesh** — Tactical mesh networking with post-quantum key exchange and routing
- **beamforming** — Adaptive beamforming control circuits for MIMO antenna arrays

## Platform Requirements

- eStream platform >= 0.12.0
- ESCIR API 1.0.0
