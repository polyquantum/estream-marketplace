# polyquantum.secure_video

Secure video distribution and content security circuits for eStream.

## Modules

- **svd** — Singular Value Decomposition for perceptual video fingerprinting
- **drm** — DRM-free content protection via lattice-based watermarking
- **streaming** — Secure streaming pipeline with proof-of-circuit attestation

## Platform Requirements

- eStream platform >= 0.12.0
- FLIR API 1.0.0
