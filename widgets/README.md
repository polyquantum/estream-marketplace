# polyquantum.widgets

UI interaction components and console widgets for eStream applications.

## Modules

- **dashboard** — Dashboard widget primitives (charts, tables, status indicators)
- **forms** — Secure form components with field-level governance
- **realtime** — Real-time data visualization widgets with streaming data feeds

## Platform Requirements

- eStream platform >= 0.12.0
- ESCIR API 1.0.0
